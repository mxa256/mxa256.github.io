# mona.md

Personal website for Mona Ascha, MD.

**Live site:** [mona.md](https://mona.md)

## About

A minimalist single-page personal site built with plain HTML and CSS. No frameworks, no build process.

## Local Development

Just open `index.html` in a browser. That's it.

## Updating Content

- **Text changes:** Edit the HTML directly in `index.html`
- **Colors:** Modify CSS variables at the top of the `<style>` block
- **Projects:** Copy an existing `.project` block and modify
